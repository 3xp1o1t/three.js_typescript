## Getting Started <a name = "getting_started"></a>

Instruccion de instalacion.

### Prerequisites

- node
- typescript
- three.js

### Installing

Como instalar e inicilizar el proyecto.

```
clone repo
cd three.js_typescript
npm install
npm run dev
```
